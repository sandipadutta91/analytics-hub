✈️ Air Cargo Analysis (SQL Project)


📌 Overview

This project analyzes airline operations using SQL to generate insights on customers, routes, and revenue. It simulates a real-world Air Cargo database to support better decision-making and improve operational efficiency.

🎯 Objectives:

Identify frequent customers for targeted offers
Analyze busiest routes for better resource allocation
Evaluate ticket sales and revenue trends


🗂️ Dataset:

->Customer – Passenger details

->Passengers_on_flights – Travel records

->Ticket_details – Booking & pricing data

->Routes – Route and distance information


⚙️ Key Skills Demonstrated:

SQL Queries (JOIN, GROUP BY, HAVING)

Aggregations & Revenue Analysis

Window Functions

Indexing & Query Optimization

Stored Procedures, Views & Cursors

Database Design (ER Diagram)



📊 Key Insights

High-value customers identified for loyalty programs

Business class generates significant revenue

Certain routes show high passenger traffic

Long-distance routes indicate premium demand

Indexing improves query performance



🛠️ Tools Used:


SQL (MySQL)

ER Modeling


🚀 Outcome

Built a structured airline database

Generated actionable business insights

Applied SQL to solve real-world problems


📌 Conclusion

This project showcases how SQL can be used to analyze data and drive business decisions in the aviation industry.
